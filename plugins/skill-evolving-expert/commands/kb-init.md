# 初始化本地知识库

为当前工作目录初始化自进化专家知识库，支持跨多个项目和工作目录。知识库架构：

- **全局总结** (~/.claude/knowledge-base): 跨项目的解决方案总结
- **本地详情** (./docs/.evolving-expert): 当前项目的详细文档

初始化脚本会自动：
1. 创建本地知识库目录结构
2. 初始化本地索引文件
3. 建立与全局知识库的链接关系
