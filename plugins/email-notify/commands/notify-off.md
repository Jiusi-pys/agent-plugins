# Disable Email Notifications

## Execution

### Step 1: Disable Notifications

```bash
python3 ~/.claude/plugins/email-notify/scripts/config_manager.py disable
```

### Step 2: Output Result

```
╔═══════════════════════════════════════════════════════╗
║         Email Notifications DISABLED                  ║
╚═══════════════════════════════════════════════════════╝

To re-enable: /notify-on
```
