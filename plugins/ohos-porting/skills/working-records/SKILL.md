# Working Records

Use this skill to keep a durable record of a porting effort.

## Record

- current phase
- latest command and result
- blockers
- produced artifacts
- next concrete step

## Helpers

```bash
./scripts/create_task.sh
./scripts/show_task.sh <task-id>
```
