# Repo Index Reference

- Refresh: `python3 tools/repo_indexer.py refresh --root <repo>`
- Preview diff: `python3 tools/repo_indexer.py scan --root <repo>`
- Partial refresh: `python3 tools/repo_indexer.py refresh --root <repo> --path src/`
- Validate artifacts: `python3 tools/repo_indexer.py doctor --root <repo>`
