# Reading Policy

Start with guides and generated summaries, not full-file reads.

1. Read `AGENTS.md`.
2. Read `docs/agent-architecture.md` or `.scanmeta/generated/repo-map.md`.
3. Read `.scanmeta/dirs/*.json` and `.scanmeta/files/*.json`.
4. Read `.scanmeta/sections/*.json` before opening large files.
5. Escalate to full-file reads only for patching, exact behavior, unresolved ambiguity, or line-level reasoning.
